# Character Count Form

A simple and interactive **Character Count Form** built using **HTML, CSS, and JavaScript**. This project allows users to type text into an input field or textarea and displays the number of characters entered in real time. It is a beginner-friendly project that demonstrates DOM manipulation, event handling, and dynamic content updates.

## Features
- Real-time character counting
- Responsive and clean user interface
- Instant updates as the user types
- Simple and beginner-friendly code
- Lightweight and easy to customize

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## Purpose
This project was created to practice HTML form elements, CSS styling, JavaScript event listeners, and DOM manipulation by building a real-time character counter.

## Future Improvements
- Add a maximum character limit
- Display remaining characters
